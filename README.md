# Financial Analysis Dashboard (Power BI)

## Project Overview
This project presents a Financial Analysis Dashboard built using Power BI to analyze company performance across different dimensions such as country, product, segment, and discount bands.

The dashboard helps stakeholders monitor key financial metrics and identify profitable areas for business growth.

## Key Metrics
- Total Sales
- Total Profit
- Cost of Goods Sold (COGS)
- Units Sold

## Dashboard Insights
The dashboard provides insights into:

• Profit contribution by different countries  
• Profit distribution across business segments  
• Impact of discount bands on profit  
• Product-wise profitability analysis  

## Tools Used
- Power BI
- Data Modeling
- DAX
- Data Visualization

## Features
- Interactive filters for year selection
- Country-wise profit analysis
- Segment performance analysis
- Product profitability comparison
- Discount band impact on revenue
